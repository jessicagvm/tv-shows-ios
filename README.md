# tv-shows-ios

The app will allow users to:
- Browse available TV shows.
- Search for shows by name.
- View details about each show and its episodes.

---

## Features

- Show list with pagination.
- Show detail view with grouped episodes per season.
- Episode detail screen.
- Search functionality.
- Custom error and empty states.
- Modular architecture with ViewModels, Mappers and Services.
- Dependency injection and mock services for previews/testing.

---

## Technologies

- Language: Swift
- Interface: SwiftUI
- Testing system: XCTest
- Software architecture pattern: MVVM 
- Network: URLSession + Concurrency + Combine for debounced search input.

---

## Notes

- Version: `v1.0.0` (stable and complete).
- Final visual polish (v1.1) to be done in next iteration.
- SwiftUI-only, compatible with iOS 16+.

---

## Getting Started

To run the project:

1. Clone the repository.
2. Open `TVShowsApp.xcodeproj` in Xcode.
3. Build and run on an simulator or device.
